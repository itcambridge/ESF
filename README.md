# Economic Symmetry Forum Website

A modern, editorial-style website inspired by the World Economic Forum design, focused on economic balance and stakeholder engagement.

## Overview

This website has been built using plain HTML, CSS, and JavaScript, following a clean, institutional design approach with card-based layouts and responsive design principles. The site is designed to attract governments and business leaders by presenting complex economic feedback loop concepts in an accessible, professional format.

## Features

- **Clean, institutional design** with generous white space and clear typography
- **Card-based layout** for centres, topics, and articles
- **Responsive design** that works seamlessly on mobile, tablet, and desktop
- **Interactive stakeholder filtering system** to explore shared concerns and goals
- **Smooth scroll navigation** for improved user experience
- **Mobile-friendly navigation** with hamburger menu for smaller screens
- **Accessible design** with proper color contrast and semantic HTML
- **CSS animations** for subtle interactive elements

## Structure

The website includes the following key sections:

1. **Hero Section**: Main headline and value proposition with call-to-action buttons
2. **Event Banner**: Information about upcoming flagship event
3. **Impact Section**: How the forum drives change through various programs
4. **Centres Grid**: Eight specialized centers that focus on different aspects of economic systems
5. **Feedback Loops Model**: Simple explanation of the four key feedback loops
6. **Latest Analysis**: Recent research articles from various centers
7. **Topic Discovery**: Quick navigation to specific topics of interest
8. **Geographies**: Regional economic analysis sections
9. **Stakeholders Table**: Interactive filtering of stakeholder groups and their concerns
10. **Newsletter Signup**: Email subscription form
11. **News Section**: External coverage of the forum's work
12. **Digital Initiatives**: Tools and platforms developed by the forum
13. **Join Section**: Call-to-action for participation
14. **Footer**: Comprehensive site navigation and legal links

## Files

- `index.html`: Main HTML structure and content
- `styles.css`: Styling and responsive design rules
- `script.js`: JavaScript functionality for interactive features

## Color Scheme

The site uses a carefully selected color palette:

- Primary cyan: `#04b6e3`
- Secondary buff: `#c5b78b`
- Ink (text): `#0b0f14`
- Paper (background): `#f6f3ea`
- Muted text: `#5b6675`

## Typography

The site uses the Inter font family (with system font fallbacks) for a clean, professional appearance. Monospace fonts are used for technical tags and labels.

## JavaScript Functionality

- Mobile navigation menu toggle
- Stakeholder filtering system
- Smooth scrolling for anchor links
- Newsletter form submission handling
- Scroll-based animations for visual appeal

## Browser Compatibility

The website is compatible with modern browsers including Chrome, Firefox, Safari, and Edge. It utilizes modern CSS features like Flexbox and Grid for layout, with appropriate fallbacks for older browsers.

To view the website, simply open the `index.html` file in a web browser. No build tools or server setup is required.
